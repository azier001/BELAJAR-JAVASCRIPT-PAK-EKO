## TAHAP 1: Persiapan Data

### Apa yang ditulis:

```javascript
const todoList = [];
```

### ðŸ“– Penjelasan:

- Membuat array kosong untuk menyimpan semua todo
- Gunakan `const` karena variabel tidak akan di-reassign (isinya boleh berubah)
- Array ini akan menjadi "database" sementara kita

### Kapan menulis:

Di paling atas file, sebelum kode lainnya

---

## TAHAP 2: Mengakses Form

### Apa yang ditulis:

```javascript
const form = document.forms['todo-form'];
```

### ðŸ“– Penjelasan:

- Mengambil elemen form dari HTML menggunakan name attribute
- `document.forms` adalah collection dari semua form di halaman
- Simpan dalam variabel agar mudah diakses berkali-kali

### Kapan menulis:

Setelah deklarasi todoList

---

## TAHAP 3: Menambahkan Event Listener

### Apa yang ditulis:

```javascript
form.addEventListener('submit', (e) => {
  
});
```

### ðŸ“– Penjelasan:

- Mendengarkan event 'submit' pada form
- Ketika user klik tombol "Add" atau tekan Enter, fungsi ini akan dijalankan
- Parameter `e` adalah event object yang berisi informasi tentang event

### Kapan menulis:

Setelah mendapatkan referensi form

---

## TAHAP 4: Mencegah Default Behavior

### Apa yang ditulis (di dalam event listener):

```javascript
e.preventDefault();
```

### ðŸ“– Penjelasan:

- Mencegah form melakukan submit default (refresh halaman)
- **PENTING:** Harus ditulis di baris pertama dalam event listener
- Tanpa ini, halaman akan refresh dan data hilang

### Kapan menulis:

Baris pertama di dalam fungsi event listener

---

## TAHAP 5: Mengambil Nilai Input

### Apa yang ditulis:

```javascript
const todo = form['todo'].value;
```

### ðŸ“– Penjelasan:

- Mengambil nilai dari input field dengan name="todo"
- `form['todo']` mengakses elemen input di dalam form
- `.value` mengambil isi text yang diketik user
- Simpan dalam variabel `todo`

### Kapan menulis:

Setelah preventDefault()

---

## TAHAP 6: Menambahkan ke Array

### Apa yang ditulis:

```javascript
todoList.push(todo);
```

### ðŸ“– Penjelasan:

- Method `push()` menambahkan item baru ke akhir array
- Item yang ditambahkan adalah nilai `todo` yang sudah diambil
- Array todoList sekarang berisi todo yang baru ditambahkan

### Kapan menulis:

Setelah mengambil nilai input

---

## TAHAP 7: Mereset Form

### Apa yang ditulis:

```javascript
form.reset();
```

### ðŸ“– Penjelasan:

- Mengosongkan semua input field dalam form
- Membuat form siap untuk input baru
- User experience: input langsung bersih setelah submit

### Kapan menulis:

Di baris terakhir event listener

---

## ðŸ” Kode Lengkap Hasil Akhir

```javascript
// TAHAP 1: Persiapan Data
const todoList = [];

// TAHAP 2: Mengakses Form
const form = document.forms['todo-form'];

// TAHAP 3-7: Event Listener lengkap
form.addEventListener('submit', (e) => {
  // TAHAP 4: Prevent default
  e.preventDefault();

  // TAHAP 5: Ambil nilai input
  const todo = form['todo'].value;

  // TAHAP 6: Tambahkan ke array
  todoList.push(todo);

  // TAHAP 7: Reset form
  form.reset();
});
```
