### **TAHAP 2: Tambahkan Form Input Todo**

#### **Langkah 2.1: Buka Tag Form**

Tulis di dalam `<body>`:

```html
<form name="todo-form" id="todo-form">
```

**ğŸ“ Penjelasan:**
- `<form>` - Tag pembuka form
- `name="todo-form"` - Nama form untuk akses via JavaScript dengan `document.forms['todo-form']`
- `id="todo-form"` - ID unik untuk seleksi dengan `document.getElementById()`

---

#### **Langkah 2.2: Buat Struktur Tabel**

```html
    <table>
        <tbody>
```

**ğŸ“ Penjelasan:**
- `<table>` - Membuat struktur tabel untuk layout form
- `<tbody>` - Body tabel, wadah untuk baris-baris data

---

#### **Langkah 2.3: Baris Pertama - Label dan Input**

```html
            <tr>
                <td><label for="todo">Todo :</label></td>
                <td><input type="text" id="todo" name="todo" autofocus="on" /></td>
            </tr>
```

**ğŸ“ Penjelasan Detail:**
- `<tr>` - Table Row (baris tabel)
- `<td>` - Table Data (sel dalam tabel)
- `<label for="todo">` - Label yang terhubung ke input dengan id="todo"
  - Ketika label diklik, input akan otomatis fokus
- `<input>` - Field input dengan atribut:
  - `type="text"` - Tipe input berupa teks biasa
  - `id="todo"` - ID untuk JavaScript dan untuk hubungan dengan label
  - `name="todo"` - Nama field saat form dikirim
  - `autofocus="on"` - Cursor otomatis masuk ke field ini saat halaman dimuat

---

#### **Langkah 2.4: Baris Kedua - Tombol Submit**

```html
            <tr>
                <td></td>
                <td><input type="submit" value="Add" /></td>
            </tr>
```

**ğŸ“ Penjelasan:**
- `<td></td>` - Sel kosong untuk alignment (agar tombol sejajar dengan input)
- `<input type="submit">` - Tombol submit dengan atribut:
  - `type="submit"` - Akan trigger event submit pada form
  - `value="Add"` - Teks yang tampil di tombol

---

#### **Langkah 2.5: Tutup Semua Tag**

```html
        </tbody>
    </table>
</form>
```

**ğŸ“ Penjelasan:**
- Tutup `</tbody>` - Akhir body tabel
- Tutup `</table>` - Akhir tabel
- Tutup `</form>` - Akhir form

**ğŸ’¡ Hasil Tahap 2:**
Form input lengkap dengan 1 field teks dan tombol Add!

**ğŸ“ Penjelasan:**

### **TAHAP 4: Tambahkan Tabel dengan Filter**

#### **Langkah 4.1: Buka Tag Tabel**

```html
<table>
```

**ğŸ“ Penjelasan:**
- `<table>` - Membuat tabel baru untuk menampilkan daftar todo

---

#### **Langkah 4.2: Bagian Header - Filter**

```html
    <thead>
        <tr>
            <th>Filter :</th>
            <th><input type="text" name="search" id="search" /></th>
        </tr>
    </thead>
```

**ğŸ“ Penjelasan Detail:**
- `<thead>` - Table Head, bagian kepala tabel
  - Biasanya digunakan untuk judul kolom atau dalam kasus ini untuk filter
- `<tr>` - Table Row dalam header
- `<th>` - Table Header (sel header yang bold secara default)
  - `<th>Filter :</th>` - Teks label untuk filter
- `<input>` di dalam `<th>` kedua dengan atribut:
  - `type="text"` - Input teks untuk pencarian
  - `name="search"` - Nama field search
  - `id="search"` - ID untuk JavaScript (akan digunakan untuk filter)

**ğŸ’¡ Fungsi:** User bisa mengetik di sini untuk filter/cari todo tertentu

---

#### **Langkah 4.3: Bagian Body - Daftar Todo**

##### **4.3.1: Buka Tag tbody**

```html
    <tbody id="todolist-body">
```

**ğŸ“ Penjelasan:**
- `<tbody>` - Table Body, wadah untuk semua baris data todo
- `id="todolist-body"` - ID penting untuk JavaScript
  - JavaScript akan menambahkan todo baru ke dalam tbody ini
  - Akan digunakan untuk manipulasi DOM (tambah/hapus todo)

---

##### **4.3.2: Baris Contoh Todo Item**

```html
        <tr>
            <td><input type="button" value="Done" /></td>
            <td>Todo Description</td>
        </tr>
```

**ğŸ“ Penjelasan Detail:**
- `<tr>` - Satu baris mewakili satu todo item
- Kolom pertama `<td>`:
  - `<input type="button">` - Tombol biasa (bukan submit)
  - `type="button"` - Tidak akan submit form, hanya trigger JavaScript
  - `value="Done"` - Teks tombol
  - **Fungsi:** Ketika diklik, akan menandai todo sebagai selesai atau menghapusnya
- Kolom kedua `<td>`:
  - `Todo Description` - Teks deskripsi todo
  - Ini adalah placeholder, nanti akan diganti dengan todo yang diinput user

**ğŸ’¡ Catatan:** Baris ini hanya contoh tampilan. Dalam aplikasi real, baris-baris akan ditambahkan dinamis dengan JavaScript

---

#### **Langkah 4.4: Tutup Semua Tag**

```html
    </tbody>
</table>
```

**ğŸ“ Penjelasan:**
- Tutup `</tbody>` - Akhir bagian body tabel
- Tutup `</table>` - Akhir tabel todo list

**ğŸ’¡ Hasil Tahap 4:**
Tabel lengkap dengan filter di header dan contoh todo item di body!

## ğŸ“ Tips Praktek

1. **Tulis pelan-pelan** - Jangan copy-paste, ketik manual untuk melatih muscle memory
2. **Perhatikan indentasi** - Gunakan 4 spasi atau 1 tab untuk setiap level nested
3. **Tutup tag dengan benar** - Setiap tag pembuka harus ada penutupnya
4. **Tes di browser** - Buka file HTML di browser setelah setiap tahap
5. **Gunakan editor yang baik** - VS Code, Sublime Text, atau Notepad++
6. **Simpan sering** - Tekan Ctrl+S setelah menyelesaikan setiap langkah
7. **Cek struktur** - Pastikan setiap tag pembuka dan penutup sejajar dengan benar

## ğŸ” Elemen-Elemen Penting

| Elemen | Fungsi | Contoh Penggunaan |
|--------|--------|-------------------|
| `<form>` | Wadah untuk input data | Mengirim data ke server atau handle dengan JS |
| `<table>` | Struktur data dalam bentuk tabel | Layout form atau tampilkan data terstruktur |
| `<thead>` | Header tabel | Judul kolom atau filter |
| `<tbody>` | Body tabel | Konten utama data |
| `<tr>` | Baris dalam tabel | Setiap baris = 1 item data |
| `<td>` | Sel data dalam tabel | Tempat konten dalam baris |
| `<th>` | Sel header (bold) | Judul kolom atau label |
| `<label>` | Label untuk input field | Meningkatkan accessibility |
| `<input>` | Field untuk input data | text, button, submit, dll |

### **Atribut Penting:**

| Atribut | Kegunaan | Contoh |
|---------|----------|--------|
| `id` | Identifikasi unik untuk JavaScript/CSS | `id="todo"` |
| `name` | Nama field untuk form processing | `name="search"` |
| `type` | Jenis input | `text`, `button`, `submit` |
| `value` | Nilai awal atau teks tombol | `value="Add"` |
| `for` | Hubungkan label ke input | `for="todo"` (sama dengan id input) |
| `autofocus` | Fokus otomatis saat load | `autofocus="on"` |
