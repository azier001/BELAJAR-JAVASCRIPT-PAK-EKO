### ğŸ”§ Cara Membuat AJAX Request

// ajax.open('GET', 'http://localhost:8080/api/hello.json');

### ğŸ“ Penjelasan Kode:

**Baris 1: Membuat Object XMLHttpRequest**
```javascript
const ajax = new XMLHttpRequest();
```
- Membuat instance baru dari class `XMLHttpRequest`
- Object `ajax` ini yang akan digunakan untuk berkomunikasi dengan server
- Simpan dalam variable `ajax` agar bisa digunakan untuk langkah selanjutnya

**Baris 2: Konfigurasi Request dengan method `open()`**
```javascript
ajax.open('GET', 'api/hello.json');
```
- Method `open()` digunakan untuk mengkonfigurasi request yang akan dikirim
- Parameter pertama: `'GET'` adalah HTTP method yang digunakan
  - `GET` digunakan untuk mengambil/membaca data dari server
- Parameter kedua: `'api/hello.json'` adalah URL endpoint yang akan diakses
  - Bisa menggunakan relative path: `'api/hello.json'`
  - Atau absolute path: `'http://localhost:8080/api/hello.json'`

**Baris 3: Mengirim Request dengan method `send()`**
```javascript
ajax.send();
```
- Method `send()` mengirim request ke server sesuai konfigurasi yang sudah dibuat
- Untuk method `GET`, tidak perlu parameter
- Request akan dikirim secara asynchronous (tidak blocking)
