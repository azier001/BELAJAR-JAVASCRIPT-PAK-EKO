## ğŸ“š Apa itu URL Search Param?

**URL Search Param** adalah cara untuk mengirim data melalui URL menggunakan **Query Parameter** (parameter pencarian). Query Parameter adalah bagian dari URL yang dimulai dengan tanda `?` dan berisi pasangan **key-value**.

### ğŸŒŸ Contoh URL dengan Query Parameter:
```
https://example.com/search?nama=John&umur=25
```
- `?` = Penanda awal query parameter
- `nama=John` = Parameter pertama
- `&` = Pemisah antar parameter
- `umur=25` = Parameter kedua

### ğŸ’¡ Keuntungan URLSearchParams:
- âœ¨ Lebih mudah dan rapi

## ğŸ› ï¸ Cara Menggunakan URLSearchParams

### Langkah 1: Membuat Object URLSearchParams
```javascript

```

### Langkah 2: Menambahkan Data dengan `.append()`
```javascript
param.append("search", "tutorial javascript");
param.append("page", "1");
```

### Langkah 3: Konversi ke String dengan `.toString()`
```javascript

// Output: search=tutorial+javascript&page=1
```

### Langkah 4: Gabungkan dengan URL
```javascript
const url = `https://example.com/api?${param.toString()}`;
console.log(url);
// Output: https://example.com/api?search=tutorial+javascript&page=1
```

## ğŸ“ Penjelasan Detail Kode

### 1ï¸âƒ£ **XMLHttpRequest()**
- Object bawaan JavaScript untuk melakukan AJAX request
- Memungkinkan komunikasi dengan server tanpa reload halaman

### 2ï¸âƒ£ **URLSearchParams()**
- Object untuk membuat dan mengelola query parameter
- Lebih mudah daripada menulis string URL manual

### 3ï¸âƒ£ **param.append(key, value)**
- Menambahkan parameter baru ke URLSearchParams
- `key` = nama parameter
- `value` = nilai parameter

### 4ï¸âƒ£ **param.toString()**
- Mengubah object URLSearchParams menjadi string
- Format: `key1=value1&key2=value2`

### 5ï¸âƒ£ **ajax.open("GET", url)**
- Menyiapkan request dengan metode GET
- URL berisi endpoint API + query parameter

### 6ï¸âƒ£ **ajax.onload**
- Callback yang dijalankan saat response berhasil diterima
- Di sini kita memproses dan menampilkan data

### 7ï¸âƒ£ **JSON.parse()**
- Mengubah JSON string menjadi JavaScript object
- Diperlukan karena response dari server berbentuk string

### 8ï¸âƒ£ **ajax.send()**
- Mengirim request ke server
- Untuk GET request, tidak perlu parameter
