# Operator Aritmatika

| Operator | Keterangan |
|----------|------------|
| + | Pertambahan |
| - | Pengurangan |
| * | Perkalian |
| ** | Exponensial |
| / | Pembagian |
| % | Sisa Bagi |

## Contoh Sederhana

```javascript
let a = 10;
let b = 3;

console.log("Pertambahan: " + (a + b));
console.log("Pengurangan: " + (a - b));
console.log("Perkalian: " + (a * b));
console.log("Exponensial: " + (a ** b));
console.log("Pembagian: " + (a / b));
console.log("Sisa Bagi: " + (a % b));
```

**Output:**
```
Pertambahan: 13
Pengurangan: 7
Perkalian: 30
Exponensial: 1000
Pembagian: 3.3333333333333335
Sisa Bagi: 1
```
