## ğŸ“Œ Ringkasan

### âœ¨ Poin Penting
- **Event** memungkinkan interaksi antara user dan aplikasi web
- Setiap element memiliki **event yang berbeda** sesuai fungsinya
- **Event Object** menyimpan informasi detail tentang event yang terjadi
- Gunakan `addEventListener()` untuk menangani event
- `event.preventDefault()` berguna untuk mencegah behavior default

### ğŸ“ Event yang Paling Sering Digunakan
1. **click** - Saat element diklik
2. **input** - Saat input berubah
3. **submit** - Saat form di-submit
4. **keydown/keyup** - Saat tombol keyboard ditekan/dilepas
5. **change** - Saat nilai input berubah (setelah blur)
6. **focus/blur** - Saat element mendapat/kehilangan fokus
7. **mouseover/mouseout** - Saat mouse masuk/keluar dari element

## ğŸš€ Tips Best Practice

### âœ… DO
- Gunakan `addEventListener()` daripada inline event handler
- Selalu tangani error yang mungkin terjadi
- Gunakan `event.preventDefault()` dengan bijak
- Beri nama function yang deskriptif untuk event handler

### âŒ DON'T
- Jangan gunakan terlalu banyak event listener pada satu element
- Jangan lupa remove event listener jika tidak dibutuhkan lagi
- Hindari inline event handler (`onclick="..."` di HTML)
