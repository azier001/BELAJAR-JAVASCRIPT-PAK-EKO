
## âš¡ Live vs Static NodeList

NodeList memiliki **2 tipe** yang berbeda dalam perilaku update-nya:

### ğŸ”´ **Live NodeList**

- Live artinya perubahan yang terjadi pada NodeList nya, **akan merubah** semua NodeList yang sama
- Saat kita menggunakan `element.childNodes`, NodeList tersebut bersifat **Live**
- Setiap perubahan DOM akan langsung tercermin pada NodeList

#### ğŸ“ Contoh Live NodeList

**HTML:**
```html
<div id="parent">
  <p>Child 1</p>
</div>
```

**JavaScript:**
```javascript
const parent = document.getElementById('parent');
const liveNodes = parent.childNodes;

console.log('Sebelum:', liveNodes.length); // 3 (text, p, text)

// Menambah node baru
const newNode = document.createElement('p');
newNode.textContent = 'Child 2';
parent.appendChild(newNode);

console.log('Sesudah:', liveNodes.length); // 4 (otomatis berubah!)
```

**Output:**
```
Sebelum: 3
Sesudah: 4
```

---

### ğŸŸ¢ **Static NodeList**

- Static artinya perubahan yang terjadi pada NodeList nya, **tidak akan merubah** semua NodeList yang sama
- Saat kita menggunakan `document.querySelectorAll()`, NodeList tersebut bersifat **Static**
- NodeList akan tetap sama meskipun DOM berubah (snapshot)

#### ğŸ“ Contoh Static NodeList

**HTML:**
```html
<div class="item">Item 1</div>
```

**JavaScript:**
```javascript
const staticNodes = document.querySelectorAll('.item');

console.log('Sebelum:', staticNodes.length); // 1

// Menambah element baru dengan class yang sama
const newItem = document.createElement('div');
newItem.className = 'item';
newItem.textContent = 'Item 2';
document.body.appendChild(newItem);

console.log('Sesudah:', staticNodes.length); // 1 (tetap tidak berubah!)

// Untuk mendapat node terbaru, query ulang
const updatedNodes = document.querySelectorAll('.item');
console.log('Setelah query ulang:', updatedNodes.length); // 2
```

**Output:**
```
Sebelum: 1
Sesudah: 1
Setelah query ulang: 2
```

## ğŸš€ Iterasi NodeList

Anda dapat melakukan iterasi pada NodeList menggunakan berbagai cara:

**HTML:**
```html
<div class="item">Item 1</div>
<div class="item">Item 2</div>
<div class="item">Item 3</div>
```

**JavaScript:**
```javascript
const nodes = document.querySelectorAll('.item');

// Cara 1: forEach (recommended)
nodes.forEach(node => {
    console.log(node.textContent);
});

// Cara 2: for...of
for (const node of nodes) {
    console.log(node.textContent);
}

// Cara 3: for loop tradisional
for (let i = 0; i < nodes.length; i++) {
    console.log(nodes[i].textContent);
}
```

**Output:**
```
Item 1
Item 2
Item 3
```
