- Saat kita membuat HTML, kadang kita membuat teks tanpa tag
- Saat halaman web tersebut direpresentasikan dalam DOM, text tanpa tag bukanlah merupakan Element, melainkan Node

- Teks "Ini adalah Text Node" dan "Ini juga Text Node" adalah **Text Node**

## âœ¨ Membuat Text Node

- Kita juga bisa membuat Text Node menggunakan method `document.createTextNode(string)`
- Hasilnya berupa Text yang bisa kita tambahkan ke Node lain

### ğŸ“‹ Contoh Kode: Membuat Text Node

```javascript
// Membuat Text Node baru
const textNode = document.createTextNode("Halo, ini adalah Text Node!");

// Menambahkan Text Node ke dalam element
const div = document.createElement("div");
div.appendChild(textNode);

// Menambahkan div ke body
document.body.appendChild(div);

console.log(textNode.nodeType); // Output: 3
console.log(textNode.nodeName); // Output: #text
console.log(textNode.nodeValue); // Output: Halo, ini adalah Text Node!
```

**Output:**
```
3
#text
Halo, ini adalah Text Node!
```

**Penjelasan:**
- `document.createTextNode()` membuat Text Node baru dengan konten string yang diberikan
- Text Node memiliki `nodeType` bernilai **3** (TEXT_NODE)
- Text Node memiliki `nodeName` bernilai **#text**
- Kita bisa menggunakan `appendChild()` untuk menambahkan Text Node ke Element

## ğŸ” Mengakses Text Node

### ğŸ“‹ Contoh Kode: Mengakses Text Node dari Element

```html

<html>
<body>
  <p id="paragraf">Ini adalah paragraf dengan teks</p>
  
  <script>
    const paragraf = document.getElementById("paragraf");
    
    // Mengakses Text Node pertama
    const textNode = paragraf.firstChild;
    
    console.log(textNode.nodeType); // Output: 3
    console.log(textNode.nodeValue); // Output: Ini adalah paragraf dengan teks
    console.log(textNode.textContent); // Output: Ini adalah paragraf dengan teks
    
    // Mengubah nilai Text Node
    textNode.nodeValue = "Teks telah diubah!";
  </script>
</body>
</html>
```

**Output:**
```
3
Ini adalah paragraf dengan teks
Ini adalah paragraf dengan teks
```

**Hasil di Browser:**
Teks paragraf akan berubah menjadi "Teks telah diubah!"

**Penjelasan:**
- `firstChild` mengakses child node pertama dari element (biasanya Text Node)
- `nodeValue` dan `textContent` bisa digunakan untuk membaca/mengubah isi Text Node
- Text Node bersifat mutable (bisa diubah nilainya)
