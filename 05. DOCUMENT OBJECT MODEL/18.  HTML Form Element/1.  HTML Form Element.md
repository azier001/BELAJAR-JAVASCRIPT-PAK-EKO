## ğŸ” Akses Form Via Name

Salah satu **kelebihan utama** Form adalah kemampuan untuk mengakses form dan input-nya hanya dengan menggunakan **name attribute**, tanpa perlu menggunakan `getElementById()` atau `querySelector()`.

### ğŸ¯ Cara Mengakses Form

- **Akses Form**: `document.forms[name]`
- **Akses Input dalam Form**: `document.forms[name][inputName]`

### ğŸ’¡ Contoh Penggunaan

```html
<!DOCTYPE html>
<html>
<head>
    <title>Akses Form Via Name</title>
</head>
<body>
    <h2>Form Login</h2>
    
    <!-- Form dengan name="loginForm" -->
    <form name="loginForm">
        <label>Username:</label>
        <input type="text" name="username" value="john_doe"><br><br>
        
        <label>Password:</label>
        <input type="password" name="password" value="secret123"><br><br>
        
        <button type="button" onclick="tampilkanData()">Lihat Data</button>
    </form>

    <script>
        function tampilkanData() {
            // Akses form menggunakan name
            const form = document.forms['loginForm'];
            
            // Akses input menggunakan name
            const username = form['username'].value;
            const password = form['password'].value;
            
            // Atau bisa juga langsung:
            const usernameAlt = document.forms['loginForm']['username'].value;
            
            alert(`Username: ${username}\nPassword: ${password}`);
        }
    </script>
</body>
</html>
```

### ğŸ“– Penjelasan Kode:

1. **`<form name="loginForm">`** - Membuat form dengan name "loginForm"
2. **`<input name="username">`** - Membuat input dengan name "username"
3. **`document.forms['loginForm']`** - Mengakses form berdasarkan name-nya
4. **`form['username'].value`** - Mengakses value dari input username
5. **Alternatif**: Bisa langsung akses dengan `document.forms['loginForm']['username'].value`

### ğŸ¬ Output:
