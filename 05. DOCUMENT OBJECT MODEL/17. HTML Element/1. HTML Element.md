## ğŸ¯ Pengertian HTML Element

**HTML Element** merupakan representasi dari elemen HTML dalam DOM (Document Object Model). Setiap tag HTML yang kita buat akan direpresentasikan sebagai objek HTMLElement di dalam JavaScript.

### ğŸ“ Contoh Sederhana

```html
<!DOCTYPE html>
<html>
<head>
    <title>Contoh HTML Element</title>
</head>
<body>
    <h1 id="judul">Halo Dunia!</h1>
    
    <script>
        // Mengakses HTML Element
        const judulElement = document.getElementById('judul');
        console.log(judulElement); // Menampilkan objek HTMLHeadingElement
        console.log(typeof judulElement); // "object"
    </script>
</body>
</html>
```

**ğŸ“¤ Output:**
```
<h1 id="judul">Halo Dunia!</h1>
object
```

**ğŸ’¡ Penjelasan:**
- `document.getElementById('judul')` mengembalikan objek HTMLElement yang merepresentasikan tag `<h1>`
- Element ini adalah objek JavaScript yang bisa kita manipulasi

## ğŸ”„ HTML Element vs Element DOM

**HTML Element** adalah turunan (inheritance) dari **Element DOM**. Ini berarti HTMLElement mewarisi semua property dan method dari Element, tetapi memiliki fitur tambahan yang spesifik untuk elemen HTML.

### ğŸ“ Contoh Perbandingan

```html
<!DOCTYPE html>
<html>
<body>
    <div id="kotak">Ini adalah div</div>
    
    <script>
        const kotak = document.getElementById('kotak');
        
        // Property dari Element DOM (parent class)
        console.log("Tag Name:", kotak.tagName); // "DIV"
        console.log("ID:", kotak.id); // "kotak"
        
        // Property dari HTMLElement (child class)
        console.log("Inner Text:", kotak.innerText); // "Ini adalah div"
        console.log("Hidden:", kotak.hidden); // false
    </script>
</body>
</html>
```

**ğŸ“¤ Output:**
```
Tag Name: DIV
ID: kotak
Inner Text: Ini adalah div
Hidden: false
```

**ğŸ’¡ Penjelasan:**
- `tagName` dan `id` adalah property dari Element DOM
- `innerText` dan `hidden` adalah property tambahan dari HTMLElement
- HTMLElement = Element + Fitur tambahan khusus HTML

## âš¡ Property dan Method Tambahan

HTMLElement memiliki **banyak sekali** property dan method tambahan yang tidak dimiliki oleh Element DOM biasa. Ini membuat manipulasi elemen HTML menjadi lebih mudah dan powerful.

### ğŸ“ Contoh Property Tambahan

```html
<!DOCTYPE html>
<html>
<body>
    <button id="tombol" style="background-color: blue; color: white;">
        Klik Saya
    </button>
    
    <script>
        const tombol = document.getElementById('tombol');
        
        // Property HTMLElement yang powerful
        console.log("Inner Text:", tombol.innerText); // "Klik Saya"
        console.log("Offset Width:", tombol.offsetWidth); // Lebar element dalam pixel
        console.log("Offset Height:", tombol.offsetHeight); // Tinggi element dalam pixel
        console.log("Content Editable:", tombol.contentEditable); // "inherit"
        
        // Mengubah property
        tombol.hidden = true; // Menyembunyikan tombol
        console.log("Tombol disembunyikan:", tombol.hidden); // true
    </script>
</body>
</html>
```

**ğŸ“¤ Output:**
```
Inner Text: Klik Saya
Offset Width: 78
Offset Height: 27
Content Editable: inherit
Tombol disembunyikan: true
```

**ğŸ’¡ Penjelasan:**
- `innerText`: Mengakses teks di dalam element
- `offsetWidth` & `offsetHeight`: Mendapatkan ukuran element
- `contentEditable`: Mengatur apakah element bisa diedit
- `hidden`: Menyembunyikan/menampilkan element

## ğŸ¨ Property vs Attribute

Kebanyakan **HTML Element** memiliki **property** yang sesuai dengan **attribute** yang ada pada tag HTML tersebut. Oleh karena itu, kita sering **tidak perlu lagi** menggunakan method `setAttribute()` dan `getAttribute()`.

### âŒ Cara Lama (Menggunakan setAttribute & getAttribute)

```html
<!DOCTYPE html>
<html>
<body>
    <input type="text" id="nama">
    
    <script>
        const inputNama = document.getElementById('nama');
        
        // Cara lama - menggunakan setAttribute & getAttribute
        inputNama.setAttribute('value', 'John Doe');
        inputNama.setAttribute('placeholder', 'Masukkan nama');
        
        console.log("Value:", inputNama.getAttribute('value')); // "John Doe"
        console.log("Placeholder:", inputNama.getAttribute('placeholder')); // "Masukkan nama"
    </script>
</body>
</html>
```

**ğŸ“¤ Output:**
```
Value: John Doe
Placeholder: Masukkan nama
```

### âœ… Cara Baru (Menggunakan Property Langsung)

```html
<!DOCTYPE html>
<html>
<body>
    <input type="text" id="nama">
    
    <script>
        const inputNama = document.getElementById('nama');
        
        // Cara baru - menggunakan property langsung (LEBIH MUDAH!)
        inputNama.value = 'John Doe';
        inputNama.placeholder = 'Masukkan nama';
        
        console.log("Value:", inputNama.value); // "John Doe"
        console.log("Placeholder:", inputNama.placeholder); // "Masukkan nama"
    </script>
</body>
</html>
```

**ğŸ“¤ Output:**
```
Value: John Doe
Placeholder: Masukkan nama
```

**ğŸ’¡ Penjelasan:**
- Attribute HTML seperti `value`, `placeholder`, `id`, `class` memiliki property JavaScript yang sesuai
- Menggunakan property **lebih cepat** dan **lebih mudah** dibaca
- Property otomatis ter-sync dengan attribute di HTML

### ğŸ“ Contoh Lengkap dengan Berbagai Property

```html
<!DOCTYPE html>
<html>
<body>
    <img id="gambar" src="photo.jpg" alt="Foto Profil" width="200">
    <a id="link" href="https://google.com">Google</a>
    <input id="checkbox" type="checkbox" checked>
    
    <script>
        // Image Element
        const gambar = document.getElementById('gambar');
        console.log("Src:", gambar.src); // URL lengkap
        console.log("Alt:", gambar.alt); // "Foto Profil"
        console.log("Width:", gambar.width); // 200
        
        // Anchor Element
        const link = document.getElementById('link');
        console.log("Href:", link.href); // URL lengkap
        console.log("Text:", link.innerText); // "Google"
        
        // Checkbox Element
        const checkbox = document.getElementById('checkbox');
        console.log("Checked:", checkbox.checked); // true
        checkbox.checked = false; // Uncheck checkbox
        console.log("Checked setelah diubah:", checkbox.checked); // false
    </script>
</body>
</html>
```

**ğŸ“¤ Output:**
```
Src: file:///path/to/photo.jpg
Alt: Foto Profil
Width: 200
Href: https://google.com/
Text: Google
Checked: true
Checked setelah diubah: false
```

**ğŸ’¡ Penjelasan:**
- Setiap jenis HTML Element (`<img>`, `<a>`, `<input>`) memiliki property spesifik
- Property ini langsung mengakses dan memodifikasi attribute HTML
- Tidak perlu `setAttribute()` atau `getAttribute()` lagi!
