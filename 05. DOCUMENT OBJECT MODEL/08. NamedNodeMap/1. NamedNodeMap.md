### ğŸ“Œ Contoh 1: Mengakses Semua Attributes

```html

<html>
<head>
    <title>NamedNodeMap Example</title>
</head>
<body>
    <div id="myDiv" class="container" data-info="test" style="color: red;">
        Hello World
    </div>

    <script>
        // Mengambil elemen
        const divElement = document.getElementById('myDiv');
        
        // Mengakses NamedNodeMap
        const attributes = divElement.attributes;
        
        console.log('Jumlah attributes:', attributes.length);
        
        // Iterasi semua attributes
        for (let i = 0; i < attributes.length; i++) {
            console.log(`${attributes[i].name} = ${attributes[i].value}`);
        }
    </script>

</html>
```

**Output:**
```

id = myDiv
class = container
data-info = test
style = color: red;
```

**Penjelasan:** 
- `element.attributes` mengembalikan objek **NamedNodeMap** yang berisi semua atribut dari elemen tersebut
- Kita dapat mengiterasi NamedNodeMap seperti array menggunakan index
- Setiap item dalam NamedNodeMap adalah objek **Attr** yang memiliki property `name` dan `value`

### ğŸ“Œ Contoh 2: Mengakses Attribute Berdasarkan Nama

```javascript
const divElement = document.getElementById('myDiv');
const attributes = divElement.attributes;

// Mengakses attribute berdasarkan nama
const classAttr = attributes.getNamedItem('class');
console.log('Class:', classAttr.value);

// Atau menggunakan bracket notation
const idAttr = attributes['id'];
console.log('ID:', idAttr.value);
```

**Output:**
```
Class: container
ID: myDiv
```

**Penjelasan:**
- Method `getNamedItem(name)` digunakan untuk mengambil attribute berdasarkan nama
- Kita juga bisa menggunakan bracket notation `attributes['namaAttribute']`
- Keduanya mengembalikan objek **Attr**

### ğŸ“Œ Contoh 3: Menambah dan Menghapus Attribute

```javascript
const divElement = document.getElementById('myDiv');
const attributes = divElement.attributes;

// Membuat attribute baru
const newAttr = document.createAttribute('data-new');
newAttr.value = 'new value';

// Menambahkan attribute
attributes.setNamedItem(newAttr);
console.log('Attribute baru ditambahkan:', attributes['data-new'].value);

// Menghapus attribute
const removedAttr = attributes.removeNamedItem('data-info');
console.log('Attribute dihapus:', removedAttr.name);
```

**Output:**
```
Attribute baru ditambahkan: new value
Attribute dihapus: data-info
```

**Penjelasan:**
- `setNamedItem(attr)` menambahkan attribute baru ke dalam NamedNodeMap
- `removeNamedItem(name)` menghapus attribute dan mengembalikan attribute yang dihapus
- Perubahan pada NamedNodeMap akan langsung mempengaruhi elemen HTML
