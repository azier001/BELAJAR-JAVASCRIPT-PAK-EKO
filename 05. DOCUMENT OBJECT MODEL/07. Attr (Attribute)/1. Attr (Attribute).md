## ğŸ› ï¸ Membuat Attr

### Cara 1: Menggunakan `createAttribute()` dan `setAttributeNode()`

#### ğŸ“Œ Langkah-langkah:
1. Untuk membuat Attr, kita bisa menggunakan `document.createAttribute(name)`
2. Lalu untuk menambahkannya ke Element, kita bisa gunakan `element.setAttributeNode(attr)`

#### ğŸ“ Contoh Kode

```html
<!DOCTYPE html>
<html>
<head>
    <title>Membuat Attr dengan createAttribute</title>
</head>
<body>
    <div id="myDiv">Hello World</div>
    
    <script>
        // Mengambil element
        const divElement = document.getElementById('myDiv');
        
        // Membuat attribute baru
        const classAttr = document.createAttribute('class');
        classAttr.value = 'container';
        
        // Menambahkan attribute ke element
        divElement.setAttributeNode(classAttr);
        
        console.log('Class attribute berhasil ditambahkan:', divElement.getAttribute('class'));
        // Output: Class attribute berhasil ditambahkan: container
    </script>
</body>
</html>
```

**Output:**
```
Class attribute berhasil ditambahkan: container
```
